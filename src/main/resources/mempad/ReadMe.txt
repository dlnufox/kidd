MemPad note taking software

>> MemPad Installation:

Important if your Windows version has restricted capabilities in the folder "Program Files" (Windows Vista/Seven):

We suggest storing portable programs like MemPad outside the "Program Files" folder. For more installation options see Help file.

>> MemPad Evaluation:

For a quick evaluation of MemPad drag&drop the "MemPad.help" file onto the MemPad.exe file. Read the Program/Installation section to get started. 

Note: The files from the "Example" subdirectory are required to test some advanced features in the help file. Please copy these files to the main directory.
