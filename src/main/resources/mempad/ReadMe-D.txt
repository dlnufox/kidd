** German **

>> MemPad Einrichtung:

Wichtig bei Nutzungseinschränkungen durch Windows im Ordner "Programme" (Vista/Seven):

Es empfiehlt sich, portable Programme wie MemPad außerhalb des Ordners "Programme" einzurichten. Weitere Möglichkeiten s. Hilfe.

>> MemPad ausprobieren:

Für einen Einstieg in die Arbeitsweise von MemPad einfach die "mempad-d.help" Datei per Drag&Drop auf die "Mempad.exe" ziehen. Zunächst den Abschnitt Programm/Einrichtung lesen.

Damit einige besondere Funktionen aus der Hilfe heraus ausprobiert werden können, sind die Dateien aus dem "Example"-Ordner erforderlich. Bitte diese Dateien ins Hauptverzeichnis kopieren. 

>> Update?: Datei "History.txt" enthält die Neuigkeiten (Englisch).
